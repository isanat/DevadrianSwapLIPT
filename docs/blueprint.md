# **App Name**: LiptonSwap

## Core Features:

- Token Purchase: Allow users to purchase LIPT tokens directly through the platform.
- Liquidity Pool: Enable users to add liquidity to a LIPT trading pair and earn fees.
- Staking Pool: Provide a staking mechanism where users can lock their LIPT tokens to earn staking rewards.
- Unilevel Referral Program: Implement a referral system that rewards users for inviting others to the platform.
- Metamask Integration: Seamlessly integrate with Metamask wallet for easy and secure transactions.
- Reward Distribution: Automatically distribute staking and referral rewards to user wallets.

## Style Guidelines:

- Primary color: Teal (#008080) for a trustworthy DeFi platform feel.
- Background color: Light teal (#E0F8F7), a desaturated shade of the primary.
- Accent color: Dark Green (#006400) to highlight key interactive elements.
- Font: 'Inter', a sans-serif font for a modern look for both headlines and body.
- Use clean and professional icons to represent different functions and features of the DeFi platform.
- Maintain a clean and intuitive layout to simplify DeFi interactions.
- Subtle animations to indicate loading states or confirm transactions.
